Seletores: definem quais elementos serão aplicados no CSS

  • Seletor por Tipo: busca elementos através de uma tag HTML específica
  • Seletor de Classe(.): busca elementos através do atributo "class"
  • Seletor de ID(#): busca elementos através do atributo ID
  • Seletor Universal(*): Aplica um elemento padrão para todas as Tags

Agrupamento de seletores

Podemos aplicar as mesmas regras CSS para vários seletores diferentes. Para isso, podemos separar os seletores utilizando uma vírgula (,)

Assim, não precisamos ficar copiando e colando os mesmo códigos para todos, nós podemos fazer um só bloco de CSS e reutilizar as mesmas regras!
